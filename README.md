# Deal Promoter

Finds genuine Amazon deals (via Keepa), validates them against the Amazon Creators API, and publishes the affiliate-tagged links to a WhatsApp channel on a schedule. A PHP/Docker monorepo, prototyped reliably.

See [docs/specs/product.md](docs/specs/product.md) for the full product spec, [docs/research/keepa.md](docs/research/keepa.md) for the Keepa API research brief, and [docs/research/creators-api.md](docs/research/creators-api.md) for the Amazon Creators API research brief.
